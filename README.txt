The Workflow Manager Flex Viewer is a fully functional application used as is. Source code is available for Flex developers if they’d like to customize the application.  

The ready to deploy Flex Viewer can be found in the dist directory.  For deployment instructions, refer to the dist\README_FlexViewer.pdf file.  This includes configuration and deployment steps, as well as a brief overview of the application architecture.  Basic knowledge of ArcGIS Server and Workflow Manager
Server REST services is assumed.
